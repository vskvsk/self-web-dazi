# 🏆 极速打字 - Vue 3 打字练习应用

一个基于 Vue 3 + TypeScript + Vite 的高性能在线打字练习应用，支持中英文打字测试、键位练习、历史记录和排行榜功能。

## ✨ 主要功能

### 🎯 打字测试
- **多语言支持**：中文/英文模式切换
- **多种文章**：精选文章库，涵盖不同难度
- **实时反馈**：准确显示打字速度(WPM)、准确率
- **统计信息**：总字符数、正确数、错误数、错误率
- **光标高亮**：当前输入位置高亮显示
- **颜色反馈**：正确(绿色)/错误(红色)字符提示
- **时间控制**：1/3/5/10/15分钟可选

### 🎮 键位练习
- **分区练习**：主键盘区、数字行、ASDF行、JKL;行、混合键位
- **虚拟键盘**：显示键盘布局，高亮目标按键
- **进度追踪**：显示当前进度和准确率
- **动画效果**：按键高亮动画

### 🏅 排行榜
- **最佳记录**：展示用户的最高 WPM 成绩
- **语言分类**：支持中文/英文独立榜单
- **详细数据**：用户名、WPM、准确率、时长、日期

### 🎨 主题系统
- **默认主题**：简洁专业的蓝色调
- **儿童主题**：活泼的彩色渐变+动画效果
  - 彩色渐变按钮
  - 浮动卡片动画
  - 弹跳/摇摆动画效果
  - 卡通字体

### 📊 历史记录
- **完整记录**：查看所有打字历史
- **数据筛选**：按语言筛选
- **统计概览**：总测试次数、平均 WPM

### ⚙️ 个性化设置
- **字体大小**：正常/大字体
- **暗黑模式**：深色主题
- **尾随光标**：显示/隐藏尾随光标
- **继续上次**：自动继续上次未完成的测试

## 🚀 快速开始

### 安装依赖

```bash

```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173

### 构建生产版本

```bash
npm run build
```


## 🎮 使用说明

1. **首次使用**
   - 输入用户名
   - 选择打字模式（中文/英文）
   - 点击"开始测试"

2. **打字测试**
   - 按照文章内容输入
   - 当前字符高亮显示
   - 完成或超时后显示成绩

3. **键位练习**
   - 选择要练习的键位区域
   - 按提示按键练习
   - 查看准确率和进度

4. **查看记录**
   - 点击"历史记录"
   - 筛选语言类型
   - 查看详细数据

## 📊 评分标准

- **WPM** = (正确字符数 / 5) / 分钟数
- **准确率** = (正确字符数 / 总字符数) × 100%
- **错误率** = (错误字符数 / 总字符数) × 100%
